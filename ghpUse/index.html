当然，我知道你肯定还不满足，下面就说说，如何把gh-pages分支clone到本地，当然，你可以在setting里面直接设置gh-pages为默认分支，但是我个人感觉这样不太友好，下面就说说另一种方式

  1. clone 到本地

    git clone https://github.com/zj1212300224/baidu-demo.github

    然后你会发现clone这是master分支的，不是想要的。

  2. 先进入项目，然后切换分支

    cd baidu-demo

    git checkout -b gh-pages origin/gh-pages

  3. 然后你发现你成功了，项目中的文件也变了，就是gh-pages分支下的文件

  4. 然后你可以做你想做的 只要记得提交就好

    git add .

    git commit -m "say sth you do" 提交到本地分支，生成一个commit对象

    git push origin gh-pages 提交到远程分支
